# nele's happiness – Relaunch

Moderne Neugestaltung der Website [neleshappiness.com](https://www.neleshappiness.com/).
Gleiche Inhalte, gleiche Bilder, gleiche Markenfarben – nur frischer, moderner und voll responsive.

## Ansehen
Einfach `index.html` im Browser öffnen (Doppelklick). Die Seite ist komplett eigenständig –
**alle Bilder sind direkt in die HTML-Datei eingebettet**, daher werden die Fotos in jedem
Browser zuverlässig angezeigt (auch direkt aus Google Drive). Keine Installation nötig,
nur für die Google-Schriften & externe Links wird Internet gebraucht.

> Der Ordner `images/` enthält die Bilder zusätzlich als Quell-/Originaldateien –
> für die Anzeige der Seite wird er aber nicht mehr benötigt.

## Aufbau
```
Neles Happiness/
├── index.html        ← die komplette Website (HTML, CSS & JS in einer Datei)
├── README.md         ← diese Datei
└── images/           ← alle Original-Bilder von der bestehenden Seite
    ├── logo.png            (Erdbeer-Logo)
    ├── hero.jpg            (Startbild)
    ├── about-nele.jpg      ("Über mich")
    ├── post-*.jpg/.png     (Blog-Bilder)
    └── *-orig.jpg          (unkomprimierte Originale in voller Auflösung)
```

## Was wurde übernommen (1:1)
- **Inhalte:** Willkommenstext, „Über mich", Ernährungsphilosophie, Blogbeiträge, Rezept-Kategorien, Kontakt.
- **Bilder:** alle Original-Fotos & das Logo von der bestehenden Seite.
- **Markenfarben** (aus dem Logo gemessen):
  - Erdbeer-Koralle `#E08080`
  - Salbeigrün `#90A070`
  - Warmes Creme `#FBF6F0`

## Was ist neu / modern
- Klares, luftiges Layout mit viel Weißraum
- Elegante Schrift-Kombi (Fraunces + Nunito Sans)
- Organische Formen, weiche Schatten, sanfte Scroll-Animationen
- Sticky-Navigation mit Blur-Effekt
- Voll responsive (Desktop, Tablet, Smartphone mit Hamburger-Menü)
- Modernes Kontaktformular (öffnet die E-Mail-App)

## Hinweise
- Navigationspunkte (Home, Über mich, Ernährung, Rezepte, Downloads, Kontakt) springen zu den
  jeweiligen Abschnitten auf der Seite (One-Page-Design).
- Rezept-Kategorien, Shop & Downloadbereich verlinken aktuell noch auf die bestehende Wix-Seite –
  diese Unterseiten können bei Bedarf im gleichen Stil ergänzt werden.
- Rechtliche Links (Impressum/Datenschutz/AGB) sind als Platzhalter gesetzt und müssen noch verknüpft werden.
